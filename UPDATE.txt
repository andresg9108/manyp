Manual to move from version v0.4.0 to v1.0.0

1. Add the following "scripts" in the package.json.

- "uglify": "grunt uglify"
- "sass": "grunt sass"
- "hbs": "grunt handlebars"

2. Modify the "aLog" array in the Gruntfile.js file as follows.

var aLog = ['./processpy.log'];

3. Modify the ".gitignore" file with the following lines.

/node_modules
/web
/src/css/dist
/src/js/dist
/src/template/dist

Ok, updated project.
