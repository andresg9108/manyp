Manual to move from version v2.0.0 to v3.0.0

1. Run "npm update".
2. Update the version.json file.
3. New file "start.js".
4. New file "server.js".
5. New folder "sys-express".
6. File: package.json

	* The following "script" items are added or modified.
		"start": "node main.js",
		"grunt": "grunt",
		"server": "node server.js",
	* New packages: 
		- npm i express --save-dev

Ok, updated project.
