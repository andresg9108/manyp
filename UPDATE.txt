Manual to move from version v1.1.0 to v2.0.0

1. Run "npm update".
2. Change version of package.json dependencies that need manual update.
3. Delete shell.js file.
4. Update the version.json file.
5. Update the LICENSE.txt file.
6. New handling in the Gruntfile.js file.
7. Now the changes will not be made in the Gruntfile.js file but in the grunt folder.

Ok, updated project.
